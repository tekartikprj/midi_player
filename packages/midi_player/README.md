# midi_player

Simple midi player (ios/android)

# Setup

```
  tekartik_midi_player:
    git:
      url: https://github.com/tekartikprj/midi_player
      path: packages/midi_player
```

## iOS 

you need to add sounds.sf2 as an asset in XCode
Simple copy sounds.sf2 in Flutter/Runner section in Xcode, it should be added as alink
